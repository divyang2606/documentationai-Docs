## Recent Releases

Track all SproutOS updates here. You receive automatic notifications in your dashboard when new versions deploy. Review changes before upgrading to ensure compatibility with your workflows.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## Added

- **Blueprint Generator:** Skip the blank-page planning stage. Generates a full website plan from a plain-text project description instantly.
- **Automation Workflows:** Automate repetitive tasks without writing code. Build multi-step automations visually with a drag-and-drop node editor.
- **Agency Analytics:** Spot which client projects are growing before they hit a limit. New dashboard section tracks project growth and scalability metrics.

## Changed

- **AI Content Model:** Get more accurate first drafts with less manual editing. Content generation accuracy improved by 25%.
- **Static Site Exports:** Spend less time waiting on builds. Export times reduced for static site exports.

## Fixed

- **Webhook Integrations:** Fixed intermittent failures during webhook execution.
- **Scale Monitoring Panel:** Fixed UI glitches causing display errors.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix", "security"]}>

## Added

- **Real-Time Collaboration:** Plan websites with your team without overwriting each other's work. Multiple members can now edit in the same live session simultaneously.
- **Custom AI Prompt Templates:** Reuse the same prompt templates across projects without rewriting them. Save and apply branded templates to any project.

## Fixed

- **API Token Security:** Patched a vulnerability in API token handling.
- **Scaling Algorithm:** Fixed errors triggered during high-traffic simulations.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Added

- **SproutOS Core:** Everything you need to build and manage websites is now in one place. Launched with four modules: Plan, Build, Automate, and Scale.
- **Natural Language Website Generation:** Turn a description into a working website without touching a page builder. Generates a complete website from plain text.

## Breaking Changes

- **Configuration Schema:** `sproutos.config.json` structure has changed. Migrate existing config files before upgrading.

</Update>

## Upgrade Instructions

Follow these steps to update SproutOS safely.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your projects and configurations.

    <Callout kind="tip">
      Use the dashboard export feature for complete backups.
    </Callout>
  </Step>
  <Step title="Update Core" icon="download">
    Run the update command for your platform.

    <CodeGroup tabs="npm,yarn">
````bash
npm update @sproutos/core@latest
````
````bash
yarn upgrade @sproutos/core@latest
````
    </CodeGroup>
  </Step>
  <Step title="Restart Services" icon="refresh-cw">
    Restart your SproutOS instance and verify workflows.
  </Step>
</Steps>
