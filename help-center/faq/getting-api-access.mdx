title: How do I get API access?
description: How to obtain your SproutOS API token.
---

## Overview

Every agency gets an API token to connect apps to SproutOS.

## Steps

1. Login to your SproutOS Dashboard
2. Go to **Settings > API Tokens**
3. Click "Generate Token"
4. Copy your new token

## Usage Example

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" https://api.sproutos.com/v1/websites
```
